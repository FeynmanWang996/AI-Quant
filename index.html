量化交易初体验：从零搭建数据引擎

一、量化交易的优势

相较于传统手工操作交易的方法，量化交易具有以下显著优势：

1. 系统性与纪律性

量化交易通过预设的数学模型和算法执行交易，完全排除了人类情绪的干扰。传统交易中，投资者容易受到恐惧、贪婪等情绪的支配，导致非理性的交易决策。而量化交易系统严格按照模型信号执行，确保交易纪律的严格执行。

2. 高效的数据处理能力

量化交易能够快速处理海量的市场数据，包括价格、成交量、财务报表、新闻舆情等多维度信息。人工分析往往受限于时间和精力，只能关注有限的信息，而量化系统可以同时监控全市场数千只股票，捕捉每一个交易机会。

3. 回测与验证

量化策略可以通过历史数据进行回测，验证策略的有效性和稳健性。投资者可以在实盘前充分了解策略在不同市场环境下的表现，从而优化参数、降低风险。传统交易方法难以进行系统性的历史验证。

4. 快速执行与低延迟

量化交易系统可以在毫秒级别完成交易决策和执行，抓住短暂的市场机会。特别是在高频交易领域，人工操作根本无法企及这样的速度。快速的响应能力对于套利策略尤为重要。

5. 风险管理与分散投资

量化系统可以精确计算风险敞口，自动进行仓位管理和止损。同时，量化策略易于实现投资组合的分散化，降低单一资产的风险。系统化的风险管理远比人工判断更为精确和及时。

6. 可重复性与可扩展性

一旦量化策略被验证有效，它可以无限次重复执行，并且容易扩展到其他市场或资产类别。一个成功的策略可以同时交易数百只股票，而人工操作难以实现这样的规模。

总结：量化交易通过系统化、自动化和科学化的方法，克服了传统交易中的情绪化、主观性和局限性，为投资者提供了更为高效、精准和可靠的交易方式。

二、基本概念解释

1. K线（蜡烛图）

定义：K线（Candlestick Chart）又称为蜡烛图或日本线，是一种用于描述证券价格波动的技术分析图表。每根K线记录了特定时间段内（如日、周、月）的开盘价、最高价、最低价和收盘价。

组成：K线由实体和影线组成。实体表示开盘价与收盘价之间的价格区间，影线表示最高价与最低价。如果收盘价高于开盘价，实体通常为红色（中国股市惯例）或空心；如果收盘价低于开盘价，实体通常为绿色或实心。

作用：K线能够直观展示市场多空双方的力量对比和价格走势，是技术分析中最基础也最重要的工具之一。通过K线形态（如锤子线、吞没形态、十字星等），投资者可以判断市场趋势和反转信号。

2. 基本面分析

定义：基本面分析（Fundamental Analysis）是通过研究影响资产内在价值的宏观经济、行业发展、公司经营等基本面因素，评估资产是否被高估或低估，从而做出投资决策的方法。

分析内容：

（1）宏观经济：GDP增长率、通货膨胀率、利率水平、货币政策、财政政策等宏观指标对整体市场的影响。

（2）行业分析：行业生命周期、竞争格局、政策支持、技术革新等行业因素对公司的影响。

（3）公司分析：财务报表分析（营收、利润、现金流、ROE、PE、PB等估值指标）、公司治理、管理团队、竞争优势等。

作用：基本面分析帮助投资者找到内在价值高于市场价格的"便宜"资产，适合中长期价值投资策略。 Warren Buffett 是基本面分析的典范。

3. 技术面分析

定义：技术面分析（Technical Analysis）是通过研究历史价格和成交量数据，运用图表和各种技术指标，预测未来价格走势的方法。技术分析认为"历史会重演"，市场价格已经反映了所有可用信息。

分析工具：

（1）K线形态：如前所述，通过K线组合判断买卖信号。

（2）趋势线：支撑线、阻力线、通道线等，识别价格趋势。

（3）技术指标：移动平均线（MA）、相对强弱指标（RSI）、MACD、布林带（Bollinger Bands）、成交量指标等。

（4）形态分析：头肩顶/底、双顶/底、三角形整理、旗形整理等价格形态。

作用：技术分析主要用于择时和短期交易决策，适合趋势跟踪、波段操作等策略。与技术分析相对的基本面分析更注重"买什么"，而技术分析更关注"什么时候买卖"。

基本面 vs 技术面：基本面分析关注"价值"，回答"这只股票值不值得买"；技术面分析关注"时机"，回答"什么时候买卖"。理想的量化策略往往结合两者，既选优质资产，又择合适时机。

（1）获取股票数据

import tushare as ts import json import urllib.request from datetime import datetime, timedelta import time # 配置Tushare Token TUSHARE_TOKEN = "***REMOVED***" API_URL = "https://api.tushare.pro" # 封装Tushare API请求函数 def tushare_query(api_name, params, fields=""): """ 通用Tushare API查询函数 参数: api_name: API接口名称，如 'daily', 'stock_basic'等 params: 参数字典，如 {'ts_code': '600048.SH'} fields: 返回字段，空字符串表示返回所有字段 返回: API响应的数据部分 """ data = { "api_name": api_name, "token": TUSHARE_TOKEN, "params": params, "fields": fields } req = urllib.request.Request( API_URL, data=json.dumps(data).encode('utf-8'), headers={'Content-Type': 'application/json'} ) try: with urllib.request.urlopen(req, timeout=30) as resp: result = json.loads(resp.read().decode('utf-8')) if result.get("code") == 0: return result.get("data") else: print(f"API错误: {result.get('msg')}") return None except Exception as e: print(f"请求失败: {e}") return None # 计算日期范围（过去一年） end_date = datetime.now().strftime("%Y%m%d") start_date = (datetime.now() - timedelta(days=365)).strftime("%Y%m%d") print(f"正在获取保利发展 (600048.SH) 从 {start_date} 到 {end_date} 的交易数据...") # 获取每日行情数据 daily_data = tushare_query( "daily", { "ts_code": "600048.SH", "start_date": start_date, "end_date": end_date }, "trade_date,open,high,low,close,vol,amount" ) if daily_data: print(f"✓ 成功获取 {len(daily_data.get('items', []))} 条交易记录") print(f"✓ 数据字段: {daily_data.get('fields')}") else: print("✗ 数据获取失败")

import csv def save_to_csv(data, filename): """ 将股票数据保存为CSV文件 参数: data: Tushare返回的数据字典 filename: 保存的文件名 """ if not data or not data.get('items'): print("没有数据可保存") return fields = data.get('fields', []) items = data.get('items', []) # 中文字段名映射 field_names_cn = { 'trade_date': '交易日期', 'open': '开盘价', 'high': '最高价', 'low': '最低价', 'close': '收盘价', 'vol': '成交量(手)', 'amount': '成交额(千元)', 'ts_code': '股票代码', 'name': '股票名称' } cn_fields = [field_names_cn.get(f, f) for f in fields] with open(filename, 'w', newline='', encoding='utf-8-sig') as f: writer = csv.writer(f) writer.writerow(cn_fields) # 写入表头 for item in items: writer.writerow(item) # 写入数据行 print(f"✓ 数据已保存到: {filename}") print(f"✓ 共保存 {len(items)} 条记录") # 保存数据 csv_filename = "600048_SH_1year.csv" save_to_csv(daily_data, csv_filename)

import matplotlib.pyplot as plt from datetime import datetime import matplotlib.dates as mdates # 设置中文字体（解决中文显示问题） plt.rcParams['font.sans-serif'] = ['Arial Unicode MS', 'SimHei', 'DejaVu Sans'] plt.rcParams['axes.unicode_minus'] = False def plot_closing_price(csv_file): """ 读取CSV数据并绘制收盘价走势图 参数: csv_file: CSV文件路径 """ dates = [] closes = [] # 读取CSV文件 with open(csv_file, 'r', encoding='utf-8-sig') as f: reader = csv.DictReader(f) for row in reader: # 转换日期格式从 YYYYMMDD 到 datetime date_str = row['交易日期'] date_obj = datetime.strptime(date_str, '%Y%m%d') dates.append(date_obj) closes.append(float(row['收盘价'])) # 数据反转（CSV是从新到旧，图表需要从旧到新） dates.reverse() closes.reverse() # 创建图表 fig, ax = plt.subplots(figsize=(14, 7)) # 绘制收盘价曲线 ax.plot(dates, closes, linewidth=2, color='#1f77b4', label='收盘价') ax.fill_between(dates, closes, alpha=0.3, color='#1f77b4') # 设置标题和标签 ax.set_title('保利发展 (600048.SH) 每日收盘价走势图 (过去一年)', fontsize=16, fontweight='bold', pad=20) ax.set_xlabel('交易日期', fontsize=12, fontweight='bold') ax.set_ylabel('收盘价 (元)', fontsize=12, fontweight='bold') # 格式化X轴日期显示 ax.xaxis.set_major_formatter(mdates.DateFormatter('%Y-%m')) ax.xaxis.set_major_locator(mdates.MonthLocator(interval=2)) plt.xticks(rotation=45, ha='right') # 添加网格 ax.grid(True, alpha=0.3, linestyle='--') ax.set_axisbelow(True) # 添加图例 ax.legend(loc='best', fontsize=11) # 统计信息 min_price = min(closes) max_price = max(closes) avg_price = sum(closes) / len(closes) latest_price = closes[-1] # 在图上添加统计信息文本框 textstr = f'最高价: {max_price:.2f}元\n最低价: {min_price:.2f}元\n平均价: {avg_price:.2f}元\n最新价: {latest_price:.2f}元' props = dict(boxstyle='round', facecolor='wheat', alpha=0.5) ax.text(0.02, 0.98, textstr, transform=ax.transAxes, fontsize=10, verticalalignment='top', bbox=props) # 调整布局并保存 plt.tight_layout() output_file = '600048_closing_price_chart.png' plt.savefig(output_file, dpi=300, bbox_inches='tight') print(f"✓ 图表已保存到: {output_file}") # 显示统计信息 print(f"\n统计信息:") print(f" 数据条数: {len(closes)} 条") print(f" 最高价: {max_price:.2f} 元") print(f" 最低价: {min_price:.2f} 元") print(f" 平均价: {avg_price:.2f} 元") print(f" 最新收盘价: {latest_price:.2f} 元") print(f" 涨跌幅: {((latest_price - closes[0]) / closes[0] * 100):.2f}%") plt.show() # 绘制图表 plot_closing_price(csv_filename)
